# maltheo.github.io

A personal site for AI/ML projects, drone-related dev projects, Blender 3D workflows, plugins, and other experiments.

## Sections

- **Projects** — Individual project pages organized by category (AI/ML, drones, Blender, plugins, misc).
- **Blog** — 100 Days of Code entries, notes, and write-ups.
- **Explorations** — Deep dives into optimization, algorithms, and technical topics.
- **Experiments** — Sandbox work, proofs of concept, and quick tests.

## Local Development

```bash
# Serve locally
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Tech

Pure HTML/CSS. No frameworks, no Jekyll. Just files.
